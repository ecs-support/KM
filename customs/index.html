ห้ามผลิต นำเข้า หรือขายถุงมือสำหรับการศัลยกรรมชนิดมีแป้ง พ.ศ. 2563

Admin
Nov 6, 2020 1 min read ประกาศกระทรวงสาธารณสุข
ประกาศกระทรวงสาธารณสุข เรื่อง ห้ามผลิต นำเข้า หรือขายถุงมือสำหรับการศัลยกรรมชนิดมีแป้ง พ.ศ. 2563

อัตราภาษีสรรพสามิต น้ำมันเชื้อเพลิงสำหรับเครื่องบินไอพ่น

Admin
Last updated on Nov 4, 2020 1 min read ประกาศกรมสรรพสามิต
รหัสสินค้าสรรพสามิตและอัตราภาษีสรรพสามิต สำหรับน้ำมันเชื้อเพลิงสำหรับเครื่องบินไอพ่น

คำอธิบายรหัส Certified Exporter Code

Admin
Last updated on Oct 28, 2020 1 min read Knowledge, สิทธิพิเศษทางการค้า
คำอธิบายรหัส Certified Exporter Code ภายใต้การรับรองถิ่นกำเนิดสินค้าด้วยตนเองของอาเซียน (ASEAN-Wide Self-Certification - AWSC)

การดำเนินกระบวนการทางศุลกากรสำหรับระบบศุลกากรผ่านแดนอาเซียน (ASEAN Customs Transit System - ACTS)

Admin
Last updated on Oct 29, 2020 1 min read พิธีการศุลกากร
ประกาศกรมศุลกากรที่ 169/.2563 เรื่อง การดำเนินกระบวนการทางศุลกากรสำหรับระบบศุลกากรผ่านแดนอาเซียน (ASEAN Customs Transit System - ACTS)

การเชื่อมโยงข้อมูลใบอนุญาต/ใบรับรองอิเล็กทรอนิกส์ ร่วมกับสำนักงานมาตรฐานผลิตภัณฑ์อุตสาหกรรม

Admin
Last updated on Oct 30, 2020 1 min read การเชื่อมโยงข้อมูลใบอนุญาต, ประกาศกรมศุลกากร
ประกาศกรมศุลกากรที่ 168/.2563 เรื่อง การเชื่อมโยงข้อมูลใบอนุญาต/ใบรับรองอิเล็กทรอนิกส์ ร่วมกับสำนักงานมาตรฐานผลิตภัณฑ์อุตสาหกรรม

รูปแบบหนังสือรับรองถิ่นกำเนิดสินค้า (Form FTA) ที่ออกจากสาธารณรัฐอินเดีย

Admin
Last updated on Oct 28, 2020 1 min read สิทธิพิเศษทางการค้า, ประกาศกองพิกัดอัตราศุลกากร
ประกาศกองพิกัดอัตราศุลากากรที่ 10/.2563 เรื่อง รูปแบบหนังสือรับรองถิ่นกำเนิดสินค้า (Form FTA) ที่ออกจากสาธารณรัฐอินเดีย

การแสดงหนังสือรับรองถิ่นกำเนิดสินค้า สำหรับผู้นำของเข้าที่ได้รับผลกระทบจากการแพร่ระบาดของโรคติดเชื้อไวรัสโคโรนา 2019 (โควิด-19)

Admin
Last updated on Oct 28, 2020 1 min read สิทธิพิเศษทางการค้า
ประกาศกรมศุลกากรที่ 166/.2563 เรื่อง การแสดงหนังสือรับรองถิ่นกำเนิดสินค้า สำหรับผู้นำของเข้าที่ได้รับผลกระทบจากการแพร่ระบาดของโรคติดเชื้อไวรัสโคโรนา 2019 (โควิด-19)

พิธีการศุลกากรสำหรับการยกเว้นอากรของตามประเภท 1 ภาค 4 (สุทธินำกลับ)

Admin
Last updated on Oct 28, 2020 1 min read การยกเว้นอากรตามภาค 4
ประกาศกรมศุลกากรที่ 163/.2563 เรื่อง พิธีการศุลกากรสำหรับการยกเว้นอากรของตามประเภท 1 ภาค 4 (สุทธินำกลับ)

แก้ไขเพิ่มเติมประกาศสำหรับ การยกเว้นอากรของตามประเภท 19 ภาค 4 (ภาชนะสำหรับบรรจุของ)
